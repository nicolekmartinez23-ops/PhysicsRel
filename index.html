Linear
y ∝ x
A straight line through the origin. If x doubles, y doubles. If x triples, y triples.
e.g. F = ma → F and a are linear (m fixed)
Squared
y ∝ x²
A curved line (parabola). If x doubles, y multiplies by 4. If x triples, y multiplies by 9.
e.g. KE = ½mv² → KE and v are squared (m fixed)
Inverse
y ∝ 1/x
A hyperbola. If x doubles, y halves. If x triples, y becomes one-third. The curve never touches either axis.
e.g. acp = v²/r → acp and r are inverse (v fixed)
Square Root
y ∝ √x
A curve that rises quickly at first, then flattens. If x quadruples, y doubles. If x multiplies by 9, y triples.
e.g. vf² = 2ad → vf and d are square root (a fixed)
How to use this guide: For each relationship type, follow the four steps to identify the relationship in a problem, predict the effect of a change, and verify your answer. A fully worked physics example is included for each type.
Linear y is directly proportional to x y ∝ x
  2. 0
    Identify
    Read the word problem carefully. Identify which variable has changed and what value it has changed to. Then identify which variable the question is asking about — this is the quantity whose new value you need to find.
    3. 1
    Recognise it
    Find the formula that includes both variables you identified in Step 0. If the variable that is changing appears on its own (no power, no square root, not under a fraction), then the relationship is linear.
    F = ma → F and a appear alone (×1 power) → Linear
  3. 2
    State the rule
    Whatever factor x is multiplied by, y is multiplied by the same factor.
    x × 2 → y × 2 x × 3 → y × 3 x ÷ 4 → y ÷ 4
  4. 3
    Apply it
    Multiply the original value of y by the same factor as x.
  5. 4
    Verify
    Substitute the new value of x directly into the formula and confirm you get the same answer.
✏ Worked Example — F = ma
A 5 kg object accelerates at 4 m/s², so F = 5 × 4 = 20 N.
The acceleration is doubled to 8 m/s². What is the new force?

Step 1 — Recognise: F = ma. a appears alone → Linear relationship between F and a.
Step 2 — State the rule: F ∝ a, so doubling a doubles F.
Step 3 — Apply: New F = 20 × 2 = 40 N.
Step 4 — Verify: F = 5 × 8 = 40 N ✓
Squared y is proportional to x² y ∝ x²
  2. 0
    Identify
    Read the word problem carefully. Identify which variable has changed and what value it has changed to. Then identify which variable the question is asking about — this is the quantity whose new value you need to find.
    3. 1
    Recognise it
    Look at the formula. If the changing variable is raised to the power of 2 (written as x² or x×x), the relationship is squared.
    KE = ½mv² → v is squared → Squared relationship between KE and v
  3. 2
    State the rule
    Whatever factor x is multiplied by, y is multiplied by that factor squared.
    x × 2 → y × 2² = y × 4 x × 3 → y × 3² = y × 9 x ÷ 2 → y ÷ 4
  4. 3
    Apply it
    Square the factor by which x changed, then multiply the original y by that squared factor.
  5. 4
    Verify
    Substitute the new value of x into the formula and confirm the result matches.
✏ Worked Example — KE = ½mv²
A 2 kg ball moves at 3 m/s, so KE = ½ × 2 × 3² = 9 J.
The speed is doubled to 6 m/s. What is the new KE?

Step 1 — Recognise: KE = ½mv². v is squared → Squared relationship between KE and v.
Step 2 — State the rule: KE ∝ v², so doubling v multiplies KE by 2² = 4.
Step 3 — Apply: New KE = 9 × 4 = 36 J.
Step 4 — Verify: KE = ½ × 2 × 6² = ½ × 2 × 36 = 36 J ✓
Inverse y is inversely proportional to x y ∝ 1/x
  2. 0
    Identify
    Read the word problem carefully. Identify which variable has changed and what value it has changed to. Then identify which variable the question is asking about — this is the quantity whose new value you need to find.
    3. 1
    Recognise it
    Look at the formula. If the changing variable appears in the denominator (bottom of a fraction), or if you can rearrange it so that x is on the bottom, the relationship is inverse.
    acp = v²/r → r is in the denominator → Inverse relationship between acp and r a = F/m → m is in the denominator → Inverse relationship between a and m
  3. 2
    State the rule
    Whatever factor x is multiplied by, y is divided by that same factor (they move in opposite directions).
    x × 2 → y ÷ 2 (y halves) x × 3 → y ÷ 3 (y becomes one-third) x ÷ 4 → y × 4 (y quadruples)
  4. 3
    Apply it
    Divide the original y by the same factor that x was multiplied by.
  5. 4
    Verify
    Substitute the new x into the formula and confirm the answer matches.
✏ Worked Example — acp = v²/r
A car rounds a bend at 6 m/s with radius 9 m, so acp = 36/9 = 4 m/s².
The radius is tripled to 27 m (speed unchanged). What is the new acp?

Step 1 — Recognise: acp = v²/r. r is in the denominator → Inverse relationship between acp and r.
Step 2 — State the rule: acp ∝ 1/r, so tripling r divides acp by 3.
Step 3 — Apply: New acp = 4 ÷ 3 = 1.33 m/s².
Step 4 — Verify: acp = 36/27 = 1.33 m/s² ✓
Square Root y is proportional to √x y ∝ √x
  2. 0
    Identify
    Read the word problem carefully. Identify which variable has changed and what value it has changed to. Then identify which variable the question is asking about — this is the quantity whose new value you need to find.
    3. 1
    Recognise it
    Look at the formula. If you need to take a square root to find the target quantity (i.e. the target is under a square root, or the other variable is squared), the relationship is square root.
    vf² = vi² + 2ad → vf = √(vi² + 2ad) If vi = 0: vf = √(2ad) → vf ∝ √d → Square root relationship
  3. 2
    State the rule
    Whatever factor x is multiplied by, y is multiplied by the square root of that factor.
    x × 4 → y × √4 = y × 2 (y doubles) x × 9 → y × √9 = y × 3 (y triples) x × 16 → y × √16 = y × 4
  4. 3
    Apply it
    Find the square root of the factor by which x changed, then multiply the original y by that square root.
  5. 4
    Verify
    Substitute the new x into the formula and confirm the answer matches.
✏ Worked Example — vf² = 2ad (vi = 0)
A ball rolls from rest with a = 2 m/s² over d = 8 m.
vf = √(2 × 2 × 8) = √32 ≈ 5.66 m/s.
The distance is quadrupled to 32 m (a unchanged). What is the new vf?

Step 1 — Recognise: vf = √(2ad). vf is found by taking a square root → Square root relationship between vf and d.
Step 2 — State the rule: vf ∝ √d, so quadrupling d multiplies vf by √4 = 2.
Step 3 — Apply: New vf = 5.66 × 2 = 11.31 m/s.
Step 4 — Verify: vf = √(2 × 2 × 32) = √128 ≈ 11.31 m/s ✓
